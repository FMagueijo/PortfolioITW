League Journey now open source on Github

Aporia - Jun 9th, 2022

League Journey, a football game inspired by the FIFA style and built in Unreal Engine, is now open source on Github.


This move allows developers and enthusiasts alike to access the game's source code, contributing to its improvement and customization.


By making League Journey open source, the development team aims to foster collaboration within the gaming community, enabling individuals to explore and enhance the game's features.
